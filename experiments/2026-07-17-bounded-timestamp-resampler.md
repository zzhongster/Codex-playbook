# 实验：有界时间戳重采样器的 60 分钟漂移与信号质量

**日期**：2026-07-17  
**项目**：macOS 录屏音频漂移 P0 spike

## 假设

一个以 host timestamp 映射目标帧位置、同时执行采样率转换和漂移校正的有界重采样器，可以在 ±1,000 ppm 输入偏差下运行 60 分钟，并把输出时长误差保持在一帧以内。

## 环境与控制变量

- 输出采样率：48,000 Hz。
- 模拟时长：3,600 秒。
- 目标总帧数：172,800,000。
- 输入 buffer：最多 4,800 帧。
- ppm 对照：`-1000, -100, -50, -10, +10, +50, +100, +1000`。
- 信号质量用例：1,000 Hz tone，+1,000 ppm，10 秒输出。

## 方法

1. 让输入设备的实际速率按指定 ppm 偏离名义 48 kHz。
2. 每个 buffer 保留其时间戳和原生帧数。
3. 用时间戳到绝对输出帧位置的映射驱动重采样，不逐块累积秒数。
4. 记录 produced frames、mapped drift、最大缓冲与信号指标。

## 数据

| ---: | ---: | ---: | ---: |
| -1000 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| -100 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| -50 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| -10 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| +10 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| +50 ppm | 172,800,000 | 172,800,000 | 0.000 ms |
| +100 ppm | 172,800,000 | 172,800,000 | 0.000 ms |

信号质量与资源：

- 一帧容差：0.020833 ms。
- 10 秒输出帧数：480,000。
- 测得频率：1,000.000 Hz。
- RMS：0.706101。
- 最大相邻样本变化：0.130806。
- 最大缓冲输入：4,800 帧。
- 总 gate：PASS。

## 结论

在实验覆盖的 ±1,000 ppm 控制范围内，绝对时间戳映射能保持精确目标帧数；重采样没有改变 1 kHz 音高，缓冲上界等于一个输入 block。

该结果证明算法容量，不证明任意硬件一定在 ±1,000 ppm 内，也不替代真实设备 timestamp continuity 验证。

## 可复用发现

- 长时漂移测试必须覆盖完整目标时长，短片无法放大 ppm 误差。
- 同时验时长、音高、幅度、相邻样本连续性和内存上界。
- 合成测试验证“能校正”，硬件 trace 验证“真实设备落在可校正范围”。

## 未覆盖范围

- 超出 ±1,000 ppm 的设备或不连续 timestamp。
- 丢 callback、设备热切换和采样率中途变化。
- USB 麦克风实机。
